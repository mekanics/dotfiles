# dotfiles

## HowTo

To install the whole thing, run this

```bash
git clone https://github.com/mekanics/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
script/bootstrap
```

## What's included

-   [x] Git config
-   [x] AppStore App
-   [x] Homebrew
-   [x] tmux settings
-   [x] vim settings
-   [ ] gpg
-   [ ] auto mount smb
-   [x] auto delete old downloads
